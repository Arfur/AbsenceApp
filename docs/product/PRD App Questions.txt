Yes — breadcrumbs should always be shown.

My Final Answer:
There should be no separate page title. The breadcrumb line replaces the page title entirely.

Move on to Page Layout next.

My Final Answer:
Option C — Hybrid (fluid for tables and dashboards, fixed for forms).


Question 5.

My Final Answer:
16px (standard spacing).

Section headers only.

My Final Answer:
No. Global UI settings should not be stored exclusively in the database.

JSON files

Use a hybrid approach. Core defaults and design‑system rules live in JSON files, with optional overrides stored in the database where appropriate.
