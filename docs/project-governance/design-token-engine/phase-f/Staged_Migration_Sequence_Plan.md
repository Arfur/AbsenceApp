## Objective
Define deterministic migration stages and sequencing dependencies.

## Stage Sequence
1. Stage 1: Baseline protection and policy readiness
2. Stage 2: Controlled coexistence enablement
3. Stage 3: Gate-driven transition progression
4. Stage 4: Retirement readiness validation
5. Stage 5: Retirement closure and stabilization

## Sequencing Rules
1. Stages execute in fixed order.
2. No stage may be skipped or merged.
3. Stage advancement requires gate pass evidence.

## Dependency Rules
1. Later stage depends on full closure evidence from prior stage.
2. Partial closure evidence is insufficient for progression.
