# Reconciliation Audit Requirements

- Phase: B5
- Date: 2026-05-15
- Status: Approved

## Mandatory Run Summary Fields
1. Run identifier
2. Schema version reference
3. Trigger class
4. Scope summary
5. State totals by classification
6. Action totals by action class
7. Exception/anomaly listing
8. Final outcome status
9. Timestamp envelope

## Audit Principles
1. Completeness: all state/action decisions must be represented.
2. Traceability: each run links to schema version and trigger context.
3. Determinism: summary format is stable across runs.
