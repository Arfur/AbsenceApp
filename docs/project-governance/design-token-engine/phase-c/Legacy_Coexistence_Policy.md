# Legacy Coexistence Policy

- Phase: C4
- Date: 2026-05-15
- Status: Approved

## Coexistence Objective
Enable controlled transition from legacy static CSS sources to schema-governed CSS outputs without nondeterministic behavior.

## Coexistence Rules
1. Schema-governed output is authoritative for transition decisions.
2. Legacy sources are retained only under explicit transition gates.
3. Dual-source ambiguity is disallowed and must be policy-resolved.

## Compliance Constraints
1. Every coexistence stage requires objective evidence.
2. Indefinite coexistence is not allowed.

- C4: Closed
